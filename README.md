# proyect_01
Mi tarea es preparar un prototipo de un modelo de machine learning para Zyfra. Esta es una empresa que desarrolla soluciones de eficiencia para la industria pesada.

El modelo que desarrolle debe predecir la cantidad de oro extraído del mineral de oro. Para este proyecto, dispongo de los datos de extracción y purificación.

El objetivo es que mi modelo ayude a optimizar la producción y a eliminar los parámetros no rentables.
Pasos:
Procesamiento de datos: Realicé una limpieza minuciosa, manejando valores nulos y eliminando anomalías con criterio.

Visualización y análisis gráfico: Mostré distribuciones claras y supe interpretar los cambios en las concentraciones a lo largo de las etapas.

Ingeniería de características: Creé variables de concentración total para validar la coherencia de los datos, lo que enriqueció el análisis.

Comparación de modelos: Evalué distintos algoritmos con validación cruzada y seleccioné el más eficaz con base en la métrica sMAPE.
